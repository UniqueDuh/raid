# RadRaveRaid
A Discord Raiding Tool written in python.
![](https://raw.githubusercontent.com/riaaaaaaaa/RadRaveRaid/master/screenie.png)

## Requirements
  - Python 3.8
  - colorama, requests

## Config Setup
  - If 'useproxy' is set to true requests will use proxies gathered from proxies.txt
